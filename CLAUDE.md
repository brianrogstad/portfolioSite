# Portfolio Site

Design and development portfolio at brianrogstad.com.

## Quick Reference

| Task | Command |
|------|---------|
| Dev server | `npm start` |
| Production build | `npm run build` |
| Unit tests | `npm test` |
| Run SSR server | `npm run serve:ssr:portfolioSite` |

## Tech Stack

- **Frontend:** Angular 18, TypeScript, SCSS, RxJS
- **SSR:** Express + Angular Universal
- **Hosting:** GitHub Pages via GitHub Actions
- **Status:** Live, stable — monitor analytics only
