After Effects Kinetic Type

Animation, Visual Effects, Design Layout, Video

A piece I created with a focus on improving my Adobe After Effects skills. Taking about four weeks to plan and create, The audio clip I used is taken from the movie Rounders. Laying the text out on different layers, I implemented the 3D layer feature to achieve a nice visual dynamic. A large percent of the motion created in this piece was done through applying animation to the camera. In other cases the layers themselves we created as well as animating 90 different masks to create reveals.

3D Chess Set

3D Modeling, Lighting, Rendering

Using Autodesk Maya I created this project focused on materials and lighting. Referenced from actual photographs I modeled these pieces in Autodesk Maya and later added textures, lighting, cameras and materials. Another aspect of the project was to create a high-resolution image. To do this I rendered separate passes and complied them into Adobe Photoshop. The breakdown included the lambert pass, shadow pass, reflections and refractions pass.
