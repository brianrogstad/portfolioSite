League of Legends Statics Application

Figma Web Applications Angular CSS Scss HTML Adobe Creative Suite UI Design Visual Design Information Architecture Frontend Development

As a gaming enthusiast and a lover of analysis and statistics, I set out to design an app that would gather API data from Riot Games, the developers of the popular e-sports game League of Legends. The app's primary aim is to provide users with a platform to filter and sort their recent games and specific champion data, enabling them to track their performance and identify their best-performing champions. This app, similar to Mobalytics or u.gg, is designed with the user in mind, with a focus on improving the layout and design of data-heavy information to enhance the user experience.
