League Index is a performance analytics tool for League of Legends players that provides personalized champion rankings based on actual gameplay data. Using a customizable composite scoring system, it analyzes match history across multiple metrics like KDA, farm, vision, and damage to help players discover which champions they truly perform best with. The application features interactive visualizations, role-based filtering, and data-driven recommendations tailored to individual playstyles rather than generic tier lists.

Salvage Auction Estimator

Task Flows Usability Testing Web Applications Angular CSS/SCSS HTML UI Design Visual Design Information Architecture Frontend Development Netlify

Salvage Auction Estimator is a bidding calculator designed for buyers at Copart and IAAI salvage vehicle auctions. It calculates all-in costs including auction fees, estimated repairs based on damage level, and transport expenses to determine maximum bid amounts at various profit margins (10%-25%). The tool helps users avoid overbidding by providing data-driven recommendations that factor in clean title values, repair methods, and total acquisition costs before placing a bid.
